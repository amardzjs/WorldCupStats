# ⚽ FIFA World Cup 2026 Live Dashboard

Experience the FIFA World Cup 2026 like never before with a sleek, futuristic, and fully responsive dashboard built with pure HTML, CSS, and JavaScript.

Powered by live tournament data, this project transforms raw match information into a beautiful interactive experience featuring real-time scores, group standings, knockout brackets, player statistics, and immersive visual effects.

## ✨ Features

🏆 Live World Cup 2026 data updates  
📊 Interactive overview dashboard  
🌎 Group standings with qualification tracking  
⚽ Match results with goals and scorers  
🥇 Top scorers leaderboard  
🎯 Team pages and tournament statistics  
🏟️ Knockout stage bracket visualization  
📈 Charts and visual analytics with Chart.js  
🌍 Country flags and venue information  
📱 Fully responsive design for desktop and mobile  
🌌 Modern glassmorphism interface with animated particles  
🔄 Automatic refresh system with multiple data sources  
⚡ Smooth transitions and optimized performance

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- Tabler Icons
- OpenFootball Data API

## 🚀 Highlights

- No frameworks required
- Single-file architecture
- Fast and lightweight
- Mobile-first experience
- Dark futuristic UI
- Live tournament statistics
- Elegant glassmorphism design
- Rich animations and interactive elements

## 📡 Live Data Source

Match data is provided by the OpenFootball World Cup repository, ensuring continuously updated tournament information.

https://github.com/openfootball/worldcup.json/blob/master/2026/worldcup.json
